<!-- GETTING STARTED -->

## Getting started

1. Prerequisites

- npm

```
npm install npm@latest -g
```

2. Installation

- Clone the repo

```
git clone https://github.com/fernancumez/fitness-api.git
```

3. cd into folder

```
cd fitness-api
```

4. Download dependencies

```
npm install
```

<!-- USAGE -->

## Usage

- Run the application `npm run start`
- Run `npm run dev` for development mode
